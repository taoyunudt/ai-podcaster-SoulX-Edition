# AI 播客生成器 - 前端使用指南

## 启动服务

```bash
cd ~/projects/ai-podcaster
python api_server.py
```

服务启动后，在浏览器中访问：

```
http://localhost:8000
```

## 界面说明

前端采用三栏布局，从左到右依次为：

### 左栏：输入内容（步骤 1）

提供三种输入方式：

1. **文本输入**
   - 直接在文本框中输入内容
   - 支持长文本

2. **网址输入**
   - 输入网址后点击"分析网址"
   - 系统会自动提取内容

3. **文档上传**
   - 支持拖拽或点击上传
   - 支持 DOCX、PDF 格式

**其他设置：**
- 播客时长滑块：3-30 分钟可调
- 主题输入：可选，留空则自动提取

点击 **"✨ 生成脚本"** 按钮开始生成脚本。

### 中栏：生成脚本（步骤 2）

显示生成的播客脚本，包括：

- **元数据**：主题、时长
- **脚本编辑器**：可直接编辑脚本内容
- **操作按钮**：
  - 🔄 重新生成：使用相同参数重新生成
  - 🎵 生成音频：根据当前脚本生成音频

### 右栏：生成音频（步骤 3）

显示生成的音频，包括：

- **元数据**：对话数量、音频时长
- **音频播放器**：直接在线播放
- **操作按钮**：
  - ⬇️ 下载音频：保存到本地
  - 🔄 重新开始：清空所有内容，重新开始

## 响应式设计

- **大屏幕（>1200px）**：三栏并排显示
- **中等屏幕（768-1200px）**：两栏显示，音频部分单独一行
- **小屏幕（<768px）**：单栏显示，上下排列
